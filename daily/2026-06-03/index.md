## 一、论文定位

## 二、核心方法

## 一、论文定位  
该工作聚焦于**序列推荐（Sequential Recommendation, SR）模型的自动调优问题**。传统上，在新数据集上对 SR 模型（如 SASRec、HSTU）进行超参搜索与训练配置，高度依赖有经验的机器学习工程师进行手工试错。VirtualMLE 旨在将这一过程自动化，利用 LLM 的推理、反思与工具使用能力，构建一个虚拟 ML 工程师代理，从而降低人工介入成本、加速模型在新数据上的适配。其价值在于**把调参经验外部化为可复用的启发式知识，并用智能体闭环替代人工调优**。

## 二、核心方法  
VirtualMLE 是一个基于 LLM‑agent 的框架，将推荐模型的优化过程组织为**执行（execution）→ 反思（reflection）→ 记忆更新（memory update）的闭环**。

- **核心模块**：  
  - **执行**：对当前配置进行试验（训练/评估），获取结果；  
  - **反思**：LLM 显式分析试验结果，形成简洁的启发式反馈；  
  - **记忆更新**：将启发式知识存入**分层记忆系统（hierarchical memory system）**，用于指导下一次决策。  
- **输出目标**：在较少的试验次数内达到具有竞争力的推荐质量（即高效率地找到较好的超参数或训练配置）。

### 4. BAHSD: Bridging the Long-tail Gap via Adaptive Distillation in Black-box Sequential Recommendation

## 一、论文定位  
该工作聚焦**黑盒序列推荐**场景下的模型提取（model extraction）问题：推荐系统常作为黑盒 API 部署，本地复现其能力的传统蒸馏方法在面对长尾分布（long-tail distribution）时失效。论文指出长尾导致严重的**信号异质性（signal heterogeneity）**——头部密集序列使教师偏好固化（teacher preference solidification），提取偏向局部模式；尾部稀疏序列则产生平坦、嘈杂的预测。现有“一刀切”式提取忽略这一差异，造成噪声过拟合和知识迁移低下。BAHSD 正是针对该信号异质性问题，提出自适应蒸馏框架，价值在于为黑盒推荐提取提供一种能弥合长尾差距（bridging the long-tail gap）的 plug-and-play 方案。

## 二、核心方法  
- **输入**：黑盒推荐 API 返回的预测信号（具体格式摘要未提供足够细节），以及相应的用户交互序列（包含头/尾不同密度的序列）。  
- **核心模块**：  
  1. **多尺度一致性探测机制（multi-scale consistency probing）**：隐式量化信号可靠性，用于区分高置信度信号与低置信度信号。  
  2. **自适应分层目标（adaptive hierarchical objective）**：  
     - 对高置信度信号，采用**动态温度 KL 散度（dynamic-temperature KL divergence）**，缓解偏好固化；

### 5. Time-Aware Diffusion based on Preference Disentanglement for Generative Recommendation

## 一、论文定位
本文针对生成式推荐（Generative Recommenders, GRs）中基于扩散模型的范式，指出其关键缺陷：现有扩散过程对用户历史交互中的所有物品施加统一的噪声和去噪，忽略了用户偏好本身具有时间非平稳性（non-stationary distribution）。论文提出 TDPM（Time-Aware Diffusion based on Preference Disentanglement）框架，将时间演化对偏好的影响显式引入语义索引（Semantic ID, SID）令牌的扩散过程，从而提升生成推荐的时序敏感性。

## 二、核心方法
TDPM 的核心思路是将用户偏好解耦为两类时间尺度上的成分：  
- **周期偏好（period preference）**：长时间跨度内保持一致的稳定兴趣；  
- **点偏好（point preference）**：由近期焦点事件触发的瞬时兴趣。  
在此基础上，模型在 SID 令牌的扩散过程中显式融合上述时间演化信号，使扩散节奏和条件与用户当前的时间上下文对齐。  
输入侧使用 SID 作为物品的表示，输出目标为生成适配用户时间状态的推荐序列。

- **提升幅度**：TDPM 在两个指标上平均分别提升 **29.21%** 和 **25.45%**。

- **最有价值的一点**：将长期稳定的兴趣与短期爆发式兴趣显式分离，并据此调节扩散过程，使得推荐既能稳定保持用户长期偏好，又能对近期行为快速响应。这非常适合兴趣演化明显、事件驱动型消费的场景（如新闻、短视频、电商大促）。
